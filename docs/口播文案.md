# 声历 VoiceCal — 项目介绍口播文案

> 时长约 90 秒 | 语速中等偏慢

---

大家好，我是声历的开发者。

声历是一款 AI 语音日程管家——用说话就能管理所有日程。

它是怎么做的呢？

第一层，自然语言理解。你说「明天下午三点开会」，它自己就能算出时间是 6 月 2 号下午三点，自动提取标题「开会」，创建日程。不需要你填任何表单。

第二层，语音交互。后台跑了一个大语言模型 Agent，接到你的指令后，自己会去调日历工具，查询、创建、删除。整个过程大概一两秒，比你自己在日历 App 里点来点去快多了。最后用 TTS 语音合成把结果读给你听。全程不用打字。

第三层，智能分流。我给它加了一个 Skill Router——常见操作走快速通道，只调一次大模型就搞定；复杂请求才走 Agent 多轮推理。既快又准。

第四层，记忆能力。它配了长期记忆，会记住你的习惯——比如你经常在哪个会议室开会，默认提醒提前多久。越用越懂你。

第五层，微信订阅推送。创建日程时开启提醒，弹出微信原生的授权弹窗，允许之后到时间会自动推送服务通知，不会错过任何安排。

技术栈方面：后端 FastAPI + AgentScope，前端 Vue 3 + uni-app 一套代码编译出微信小程序和 H5 两端，数据库 PostgreSQL + Redis，向量检索用 Milvus + Mem0，部署用 Docker Compose。

做这个项目的初衷很简单——我觉得现有的日历 App 交互太重了，语音才是最自然的输入方式。感谢大家的时间。

## 短版（30 秒电梯演讲）

声历，一款 AI 语音日程管家。说话就能管理日程——创建、查询、删除、提醒，全流程语音交互。

后端搭载大语言模型 Agent，能理解自然语言时间表达，自动调用日历工具，TTS 语音回复。一个 Skill Router 分流架构保证速度，长期记忆让 Agent 越用越懂你。

一套代码跨微信小程序和 H5，现在就可以扫码体验。感谢关注。
