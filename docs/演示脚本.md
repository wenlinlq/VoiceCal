# 声历 VoiceCal 演示脚本

> 建议时长：2-3 分钟 | 录制工具：微信开发者工具 + 系统录屏

---

## 准备

- [ ] 微信开发者工具打开 `dist/build/mp-weixin`
- [ ] 真机预览或模拟器，确保麦克风权限已开
- [ ] 系统录屏工具就绪（QuickTime / OBS）

---

## 开场（10秒）

> 声历，一款 AI 语音日程管家。说话就能管日程。

画面：首页日历视图

---

## 场景一：创建日程（40秒）

> "帮我创建一个明天下午三点的项目会议"

1. 点麦克风按钮
2. 说话："明天下午三点开会讨论产品方案"
3. 画面展示：ASR 实时显示识别文字「思考中...」
4. Agent 回复语音："已添加，确认创建吗？"
5. 说"确认" → Agent："✅ 已安排"
6. 画面展示：日历上出现新日程 ✅

**亮点**：自然语言理解、确认机制、语音+文字双通道

---

## 场景二：查询日程（25秒）

> "今天有什么安排"

1. 点麦克风
2. 说话："今天有什么安排"
3. Agent 语音回复："今天有 N 个安排：上午九点..."
4. 画面展示：语音播报时同时显示文字

**亮点**：多模态反馈、快速查询

---

## 场景三：删除日程（25秒）

> "删除明天下午的会"

1. 说话："删除明天下午三点的项目会议"
2. Agent："明天下午三点项目会议，确认删除吗？"
3. 说"确认" → Agent："已删除"
4. 画面展示：日历上日程消失，列表刷新

**亮点**：模糊匹配、安全确认、实时同步

---

## 场景四：订阅推送（30秒）

> 创建带提醒的日程，演示微信授权和推送

1. 通过日历界面创建一个日程
2. 打开「提醒」开关 → 保存
3. 弹出微信原生订阅授权弹窗 → 特写
4. 点击「允许」
5. 说明："到这个时间，微信会自动推送服务通知"

**亮点**：微信原生授权、定时推送、emoji 标题

---

## 收尾（10秒）

> 声历，让日程管理像说话一样简单。

画面：首页 + Logo
