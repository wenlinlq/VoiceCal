## 项目亮点

### 1. 全链路语音交互
用户说话 → ASR 转录 → Agent 理解意图 → 执行操作 → TTS 语音回复 → 前端播放。语音+文字双通道反馈。

### 2. Skill Router 分流架构
自研意图前置分流层。常见操作 **1 次 LLM 调用**直接执行，复杂操作降级 ReActAgent。比纯 Agent 循环快 2-3 倍。

```
用户输入 → 意图分类(1次LLM)
  ├── 创建/查询/删除 → 直接调工具执行 ← 80% 走这里
  └── 修改/闲聊/低置信 → ReActAgent ← 兜底
```

### 3. 智能时间解析
LLM 直接输出 ISO 8601，支持「明天下午三点」「下周二」「6月2号」「大后天」「半小时后」等，无需正则。

### 4. 长期记忆
Mem0 + Milvus 向量数据库。Agent 记住用户习惯，跨会话持久化，越用越懂。

### 5. 微信订阅推送
创建日程开提醒 → 微信原生授权弹窗 → 到时间自动推送服务通知。emoji 标题自适应。

### 6. 多用户 Session 隔离
Redis 按 `(user_id, session_id)` 隔离，独立 TTL、对话上下文、WebSocket 连接。Redis 不可用时自动降级。

### 7. 跨平台一套代码
Vue 3 / uni-app → 微信小程序 + H5。条件编译分离平台逻辑，录音、播放、画布各自独立。

## 功能

|------|------|
| 🎙️ 语音创建 | "明天下午三点开会" → 提取时间标题 → 创建 |

## 使用方法

### H5 浏览器
1. 打开 [https://latekin.jufu.vip](https://latekin.jufu.vip) 或本地 `http://localhost:5173`
2. 登录页点击「以七牛云老师身份进入」
3. 点麦克风 → 说话 → Agent 回复

### 微信小程序
1. 微信开发者工具导入 `frontend/dist/build/mp-weixin`
2. 真机预览 → 自动微信登录 → 说话交互
3. 创建日程开提醒 → 弹出订阅授权

> 小程序因 ICP 备案审核中暂未正式上线，可通过开发版/体验版使用。
